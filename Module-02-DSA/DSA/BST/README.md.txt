# Binary Search Tree (BST)

## Aim

To create a Binary Search Tree, insert elements, perform Inorder Traversal, and search for an element.

## Theory

A Binary Search Tree (BST) is a binary tree in which:
- Left subtree contains smaller values.
- Right subtree contains larger values.

### Operations

- Insert
- Search
- Inorder Traversal

## Time Complexity

| Operation | Average | Worst |
|----------|---------|-------|
| Insert | O(log n) | O(n) |
| Search | O(log n) | O(n) |
| Traversal | O(n) | O(n) |

## Sample Output

Inorder Traversal:
20 30 40 50 60 70 80

40 found in BST.
