# Selection Sort

## Aim

To sort an array using the Selection Sort algorithm.

## Theory

Selection Sort repeatedly finds the smallest element from the unsorted part of the array and places it at the beginning of the sorted part.

## Algorithm

1. Start from the first element.
2. Find the smallest element in the remaining array.
3. Swap it with the current element.
4. Repeat until the array is sorted.

## Time Complexity

- Best Case: O(n²)
- Average Case: O(n²)
- Worst Case: O(n²)

## Space Complexity

O(1)

## Sample Output

Original Array:
64 25 12 22 11

Sorted Array:
11 12 22 25 64
