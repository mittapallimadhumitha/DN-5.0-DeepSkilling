# Bubble Sort

## Aim

To sort an array using the Bubble Sort algorithm.

## Theory

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. After each pass, the largest element moves to its correct position.

## Algorithm

1. Start from the first element.
2. Compare adjacent elements.
3. Swap them if they are in the wrong order.
4. Repeat until the array is sorted.

## Time Complexity

- Best Case: O(n)
- Average Case: O(n²)
- Worst Case: O(n²)

## Space Complexity

O(1)

## Sample Output

Original Array:
64 34 25 12 22 11 90

Sorted Array:
11 12 22 25 34 64 90
