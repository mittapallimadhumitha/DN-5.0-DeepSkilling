# Lab 1 - Understanding ORM with a Retail Inventory System

## Objective
Understand how Entity Framework Core maps C# classes to SQL Server database tables using Object Relational Mapping (ORM).

## What is ORM?
Object Relational Mapping (ORM) is a technique that maps C# classes and objects to relational database tables, allowing developers to work with data using objects instead of writing SQL queries manually.

## Benefits of ORM
- Reduces SQL code
- Improves productivity
- Easier maintenance
- Object-oriented programming support

## EF Core vs EF Framework

### EF Core
- Cross-platform
- Lightweight
- Supports LINQ
- Supports asynchronous queries
- High performance

### EF Framework
- Windows only
- More mature
- Less flexible
- Larger framework

## EF Core 8 Features
