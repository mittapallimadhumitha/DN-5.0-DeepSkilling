# First Web API using .NET Core

## Objective

Create a simple ASP.NET Core Web API with CRUD action methods.

## HTTP Action Verbs

- GET
- POST
- PUT
- DELETE

## API Endpoint

http://localhost:5290/api/values

## GET Output

```json
[
  "Apple",
  "Laptop",
  "Rice Bag"
]
```

## Project Structure

- Controllers
- Program.cs
- appsettings.json
